## Session 1 – Introduction to Microsoft Excel

### What is Microsoft Excel?

Microsoft Excel is a spreadsheet software developed by Microsoft that enables users to organize, calculate, analyse and visualize data efficiently.

Excel is widely used in:

- Civil Engineering
- Structural Engineering
- Quantity Surveying
- Transportation Engineering
- Project Management
- Construction Planning
- Research and Data Analysis

---

## Applications of Excel in Civil Engineering

Excel is commonly used for:

- Quantity Estimation
- Material Consumption Calculations
- Laboratory Test Data Analysis
- Survey Data Processing
- Preparation of Bar Charts
- Cost Estimation
- Preparation of Reports
- Data Visualization
- Research Data Analysis

# Session 3 – Workbooks and Worksheets

### Workbook

A workbook is an Excel file that contains one or more worksheets.

Example:

```
Student_Record.xlsx
```

---

### Worksheet

A worksheet is an individual spreadsheet inside a workbook.

Example:

```
Sheet1
Sheet2
Sheet3
```

# Session 4 – Cell Addressing

Every cell has a unique address.

Examples

| Column B Row 5 | B5 |
| Column D Row 10 | D10 |

# Session 7 – Basic Formulas

Every formula begins with

```
=
```

Examples

Addition

```
=A2+B2
```

Subtraction

```
=A2-B2
```

Multiplication

```
=A2*B2
```

Division

```
=A2/B2
```

---

# Relative and Absolute References

## Relative Reference

```
=A2+B2
```

Changes automatically when copied.

---

## Absolute Reference

```
=$A$2
```

Does not change when copied.

---

# Session 8 – Basic Functions

## SUM()

Adds values together.

Example

```
=SUM(B2:B10)
```

---

## AVERAGE()

Calculates average.

Example

```
=AVERAGE(B2:B10)
```

---

## MAX()

Returns largest value.

Example

```
=MAX(B2:B10)
```

---

## MIN()

Returns smallest value.

Example

```
=MIN(B2:B10)
```

---

## COUNT()

Counts numerical entries.

Example

```
=COUNT(B2:B10)
```

---

## ROUND()

Rounds a number.

Example

```
=ROUND(B2,2)
```
