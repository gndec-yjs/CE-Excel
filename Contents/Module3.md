## Session 1 – Introduction to Data Visualization

### Why Visualize Data?

Data visualization transforms numerical data into graphical form, making it easier to:

- Identify trends
- Compare values
- Detect patterns
- Present results effectively
- Support engineering decision-making

---

## Applications in Civil Engineering

Excel charts are commonly used for:

- Laboratory Test Results
- Concrete Strength Comparison
- Material Consumption
- Survey Data
- Project Progress
- Cost Analysis
- Research Data
- Quality Control

---

# Session 2 – Types of Charts

Excel provides several chart types depending on the nature of the data.

---

## 1. Column Chart

### Best Used For

- Comparing values
- Laboratory results
- Material quantities

### Example

| Cube No. | Strength (MPa) |
|----------|---------------:|
| 1 | 31 |
| 2 | 33 |
| 3 | 30 |
| 4 | 35 |

---

## 2. Bar Chart

### Best Used For

- Comparing categories
- Horizontal comparison of data

Example:

Comparison of material costs.

---

## 3. Line Chart

### Best Used For

- Showing trends over time
- Monthly progress
- Temperature variation
- Settlement monitoring

Example:

Concrete strength development over 7, 14 and 28 days.

---

## 4. Pie Chart

### Best Used For

Showing percentage contribution.

Example:

Material Cost Distribution

- Cement
- Sand
- Aggregate
- Steel

---

## 5. Scatter Plot (XY Chart)

### Best Used For

Engineering relationships between two variables.

Examples:

- Load vs Deflection
- Stress vs Strain
- Time vs Settlement

# Session 4 – Introduction to Pivot Tables

## What is a Pivot Table?

A Pivot Table is a powerful Excel feature used to summarize, organize, and analyse large datasets without changing the original data.

Instead of manually calculating totals or averages, Pivot Tables generate summaries automatically.

---

## Applications

- Student Performance Analysis
- Laboratory Test Summary
- Material Consumption
- Cost Analysis
- Attendance Reports
- Sales Reports

---

# Creating a Pivot Table

Steps:

1. Select the dataset.
2. Click **Insert → Pivot Table**.
3. Choose the data range.
4. Drag fields into:
   - Rows
   - Columns
   - Values
   - Filters

# Session 6 – Goal Seek (Introduction)

## What is Goal Seek?

Goal Seek is a What-If Analysis tool used to determine the required input value to achieve a desired output.

---

### Example

Suppose a student has scored:

- Internal Marks = 24

The student wants an overall score of:

```
75
```

Goal Seek can calculate the required external examination marks.

---

## Applications

- Engineering calculations
- Cost estimation
- Profit calculation
- Target percentage
- Material optimization
